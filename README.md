# Install
- copy the cloned in `.rmp/plugins` folder

# Configurations
```lua
{
    "rose-pine-rmp",
    -- config
    {
        VARIANT = "main" --"main" "dawn" "moon" 
    }
}
```
- disable theme from settings config
